*Beat The Dot Save Settings*

Save your settings so you don't have to type parameters for `p` command, it will load from your saved settings.

Usage:

```
.p-save -u <user> -p <pass>
```

Available parameters:

```
-u <user>      ex: -u 1
-p <pass>      ex: -p 1234
-a <lunchAt>   ex: -a 11:30
-l <lunchTime> ex: -l 60
-t <tolerance> ex: -t 10
-w <workHours> ex: -w 8
-s             ex: -s
```
