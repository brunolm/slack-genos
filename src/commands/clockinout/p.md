*Beat The Dot*

Checks when you have to clock in/out.

Usage:

```
.ponto -u <user> -p <pass>
```

Available parameters:
```
-u <user>      ex: -u 1
-p <pass>      ex: -p 1234
-a <lunchAt>   ex: -a 11:30
-l <lunchTime> ex: -l 60
-t <tolerance> ex: -t 10
-w <workHours> ex: -w 8
-m <MM-YYYY>   ex: -m 05-2016
-s             ex: -s (shows time grid)
-v             ex: -v (shows time grid with corrections)
```

See also: `p-save`
